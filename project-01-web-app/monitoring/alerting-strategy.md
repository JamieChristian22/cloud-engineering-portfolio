# Alerting Strategy

## Purpose

Alerting helps cloud teams detect issues before they become major outages.

## Recommended Alerts

| Alert | Reason |
|---|---|
| High CPU | Detect compute pressure |
| VM unavailable | Detect infrastructure outage |
| Apache down | Detect application outage |
| High network errors | Detect connectivity problems |
| Failed health check | Detect application unavailability |

## Future Implementation

Implement CloudWatch Alarms, Azure Monitor Alerts, and Google Cloud Alerting Policies.
