# Security Recommendations

## Recommended Improvements

The current project is portfolio-ready, but a production-grade version should include additional hardening.

## Improvements

1. Add HTTPS/TLS certificates.
2. Redirect HTTP to HTTPS.
3. Restrict SSH access more aggressively.
4. Use a bastion host or session manager.
5. Add WAF protection.
6. Add IAM roles and managed identities.
7. Enable alerts for suspicious activity.
8. Centralize logs.
9. Add vulnerability scanning.
10. Automate infrastructure with Terraform.

## Final Security Assessment

The current security design is appropriate for a controlled portfolio deployment and demonstrates strong awareness of cloud-native security concepts.
