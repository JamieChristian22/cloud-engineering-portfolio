# Project Overview

## Project Name

Multi-Cloud Web Application Platform

## Purpose

This project demonstrates how to deploy, secure, monitor, and document a web application across AWS, Azure, and GCP.

## What Was Built

A static Apache web application was deployed on three cloud-hosted virtual machines.

## Why It Matters

This project shows practical cloud engineering skill across the three major cloud providers. It demonstrates hands-on ability in compute, networking, security, monitoring, Linux, and documentation.
