# GCP Network Design

## Services

- Google Cloud VPC
- Compute Engine network interface
- External IP address
- Firewall rules

## Firewall Rules

| Port | Protocol | Purpose |
|---:|---|---|
| 22 | TCP | SSH administration |
| 80 | TCP | HTTP web access |
| 443 | TCP | HTTPS readiness |

## Outcome

The Compute Engine VM was accessible through an external IP address and protected by GCP firewall rules.
