Multi-Cloud Web Application Platform

AWS • Microsoft Azure • Google Cloud Platform

Built by Jamie Christian II

This application demonstrates cloud infrastructure deployment across the three major public cloud providers. The purpose of the project is to show practical cloud engineering ability through compute deployment, Linux administration, firewall configuration, monitoring validation, and professional documentation.

Cloud Providers

  • Amazon Web Services using EC2
  • Microsoft Azure using Azure Virtual Machines
  • Google Cloud Platform using Compute Engine

Engineering Work Demonstrated

  • Virtual machine deployment
  • Apache web server installation
  • Public endpoint validation
  • Cloud firewall configuration
  • Monitoring and logging review
  • Security and runbook documentation
