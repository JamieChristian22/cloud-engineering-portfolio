# Access Model

## Platform Admins

Manage the shared services platform and governance controls.

## Cloud Engineers

Deploy and operate shared services.

## Security Team

Review secrets, logs, IAM roles, and compliance controls.

## Auditors

Review evidence, configuration, and logs.

## Application Teams

Consume shared platform services without directly managing them.

## Best Practices

- Enforce MFA
- Avoid long-term access keys
- Use roles instead of users where possible
- Perform regular access reviews
