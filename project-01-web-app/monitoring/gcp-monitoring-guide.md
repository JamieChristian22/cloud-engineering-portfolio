# GCP Monitoring Guide

## Purpose

Google Cloud Monitoring and Cloud Logging were used to observe the Compute Engine VM.

## Metrics Reviewed

- CPU usage
- Network traffic
- Disk activity
- Instance health
- Logs Explorer activity

## Why These Metrics Matter

Monitoring confirms VM health and performance. Logging helps troubleshoot activity and platform events.

## Validation

The GCP monitoring and logging screenshots prove that the VM was visible in Google Cloud operations tools.
