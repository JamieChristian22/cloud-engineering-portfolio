# Amazon Elastic Container Registry

## Purpose

Amazon ECR is a private container registry used to store and manage Docker images.

## Repository

Name: `enterprise-shared-services`

Configuration:
- Private repository
- Scan on push enabled
- AWS-managed encryption
- Resource tags applied

## Enterprise Value

ECR provides private container storage, IAM-controlled image access, image scanning, CI/CD integration, and ECS/EKS/Lambda container deployment support.
