# Requirements

## Functional Requirements

- The web page must load from AWS.
- The web page must load from Azure.
- The web page must load from GCP.
- Apache must run on all deployed VMs.
- Public IP access must be validated.

## Non-Functional Requirements

- Documentation must be clear.
- Screenshots must prove work was completed.
- Security controls must be documented.
- Monitoring must be reviewed.
- Cost control must be considered.
