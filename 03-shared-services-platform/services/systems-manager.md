# AWS Systems Manager

## Purpose

AWS Systems Manager provides secure remote administration and operational management for EC2 instances.

## What Was Built

- IAM role for Systems Manager access
- Managed node registration
- Session Manager browser-based shell access

## Enterprise Value

Systems Manager reduces need for bastion hosts, avoids public SSH exposure, supports patching, supports inventory, supports run command automation, and improves operational security.
