# Lessons Learned

## Technical Lessons

- Shared Services VPCs centralize platform capabilities.
- Systems Manager reduces the need for bastion hosts.
- Secrets Manager prevents hardcoded credentials.
- ACM centralizes certificate lifecycle management.
- ECR supports secure container image storage.
- CloudWatch Log Groups support centralized logging.
- AWS Backup supports disaster recovery planning.

## Portfolio Lessons

This project is strong because it demonstrates reusable infrastructure and internal platform design instead of only basic resource deployment.
