# GCP Deployment Guide

## Steps

1. Created Compute Engine VM.
2. Selected E2 machine type.
3. Enabled HTTP and HTTPS traffic.
4. Verified firewall rules.
5. Connected through browser SSH.
6. Installed Apache.
7. Deployed custom HTML page.
8. Reviewed Cloud Monitoring and Cloud Logging.

## Validation

The custom web page loaded successfully through the Compute Engine external IP.
