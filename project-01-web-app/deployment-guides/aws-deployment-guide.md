# AWS Deployment Guide

## Steps

1. Created EC2 instance.
2. Selected Amazon Linux.
3. Configured Security Group for SSH, HTTP, and HTTPS.
4. Connected using SSH.
5. Installed Apache.
6. Deployed custom HTML page.
7. Verified public access.
8. Reviewed CloudWatch metrics.

## Validation

The custom web page loaded successfully through the EC2 public IPv4 address.
