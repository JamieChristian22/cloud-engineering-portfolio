# AWS Certificate Manager

## Purpose

AWS Certificate Manager provides centralized SSL/TLS certificate management for AWS resources.

## Enterprise Uses

- Application Load Balancers
- API Gateway
- CloudFront
- Internal HTTPS services
- Secure web applications

## Lab Scope

This project demonstrates the ACM certificate request workflow using DNS validation. In production, the organization would validate the domain using DNS records.

## Benefits

- Automatic renewal for supported certificates
- Centralized management
- Native AWS integration
- Reduced manual certificate handling
